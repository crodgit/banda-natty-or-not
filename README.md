# Criação de videoclipe utilizando IA

## 📒 Descrição
Esse projeto ajudou a demonstrar, de forma dinâmica e irreverente, o processo de exploração que algumas categorias sofrem no mercado de trabalho que, no caso, são os arqueólogos. 

## 🤖 Tecnologias Utilizadas
Foi utilizado o ChatGPT para criação da letra, o SUNO para a criação do audio e melodia, o Canva para criação do Avatar e o D-ID para criação do videoclipe.

## 🧐 Processo de Criação
No prompt do ChatGPT, informei a descrição da letra que gostaria que fosse gerada. No Suno, incluí a letra e indiquei o tipo de música. No Canva solicitei que fosse gerada uma imagem de uma mulher, nos moldes de cantora sertaneja, e no D-ID incluí o avatar e incluí o áudio gerado pelo SUNO.
